# ✍️ 投稿指南（保姆级教程）

想分享你的经验？跟着下面一步步做，5分钟搞定。**不需要任何技术背景。**

---

## 🎯 第一步：打开文章文件夹

在电脑上打开这个文件夹：

> **`docs\share\`**

你在项目里就能找到它 👇

```
fjango\
  ├── docs\
  │   └── share\          ← 就是这个文件夹！点进去
  │       ├── index.md
  │       └── sample-article.md   ← 示例文章，可以看看长啥样
```

---

## ✏️ 第二步：新建你的文章文件

1. 在 `share` 文件夹里，右键点击 `sample-article.md`
2. 选择 **复制**
3. 在空白处右键，选择 **粘贴**
4. 把副本的名字改成你的文章标题（用英文或拼音）
   - 比如：`how-i-learn-coding.md` 或 `my-experience.md`
   - ⚠️ 名字不要有空格，用 `-` 代替空格

---

## 📝 第三步：写你的文章

1. 右键你新建的文件 → **打开方式** → **记事本**
2. 把内容改成你自己的，照着下面的格式写：

```markdown
# 你的文章标题

!!! info "作者信息"
    - **作者**：你的名字或网名
    - **时间**：2026年X月X日
    - **关键词**：学习方法、效率

## 背景

简单说说你为什么要写这篇分享。

## 正文

这里是你的经验内容。想到什么写什么就行。

### 小标题（可选）

可以分小节来写，加上 `###` 就是小标题。

## 总结

一两句话总结你想说的。

---

*本文采用 CC BY-SA 4.0 许可*
```

> 💡 **格式说明：**
> - `#` 开头 = 大标题
> - `##` 开头 = 中标题
> - `###` 开头 = 小标题
> - 普通文字直接写就行
> - 换行空一行就是新段落



## 📩 第四步：把文章发给我

文章写好后，**把文件名告诉我**就行。

比如跟我说：

> "我写了一篇文章，文件名是 `how-i-learn-coding.md`"

剩下的事情交给我——我会帮你：
- ✅ 把文章加到网站导航栏
- ✅ 推送到 GitHub
- ✅ 网站自动更新，1-2分钟后所有人都能看到



## 📋 写文章的小建议

| ✅ 推荐 | ❌ 避免 |
|---------|---------|
| 写自己的真实经历 | 复制别人的内容 |
| 口语化，像在聊天 | 写成论文风格 |
| 可以提到具体的人和事 | 暴露别人隐私信息 |
| 分享成功经验和失败教训 | 只说好的不说坏的 |



## 🖼️ 想在文章里加图片？

把你截图/图片放到 `docs\assets\` 文件夹里，然后在文章里这样写：

```markdown
![图片描述](../assets/你的图片名.png)
```

告诉我你放了图片，我帮你也一起处理。



## ❓ 常见问题

**Q：.md 文件是什么？**
A：跟 .txt 一样是纯文字文件，用记事本就能打开编辑。只是它多了点格式标记（像 `#`、`##` 这些）。

**Q：我不懂 Markdown 语法怎么办？**
A：没关系！基本上当普通文字写就行。记住 `#` = 标题，其他直接写。

**Q：我写错了能改吗？**
A：随时可以修改，改完告诉我一声，我重新帮你推送。

**Q：我能删掉我的文章吗？**
A：当然可以，告诉我要删哪篇就行。
