# ✍️ 投稿指南

想让你的经验帮助到更多人吗？欢迎投稿！

---

## 方式一：📧 邮箱投稿（推荐新手）

把你想分享的经验写好后，发送到：

> **投稿邮箱：** `你的邮箱@example.com`

请包含以下信息：
- 文章标题
- 作者署名（可用网名）
- 正文内容

我们会帮你整理并发布到网站上。

---

## 方式二：🔧 GitHub 直接投稿

如果你熟悉 Git 操作：

1. **Fork** 本仓库
2. 在 `docs/share/` 目录下创建你的 `.md` 文件
3. 在 `mkdocs.yml` 的 `nav` 部分添加你的文章链接
4. 提交 **Pull Request**

---

## 📝 投稿模板

你可以参考以下模板来写你的经验分享：

```markdown
# 文章标题

!!! info "作者信息"
    - **作者**：你的名字/网名
    - **时间**：2026年X月
    - **关键词**：标签1、标签2

## 背景

简单介绍你的情况和为什么要写这篇分享。

## 正文

详细的经验内容。可以分小节展开。

## 总结

一两句话总结你想传达的核心信息。

---

*本文采用 CC BY-SA 4.0 许可*
```

---

## ✅ 内容规范

- 内容真实、原创
- 不涉及违法或不当信息
- 尊重他人隐私（可对敏感信息做脱敏处理）
- 友善表达，不传播焦虑

---

*期待你的分享！每一篇真诚的文字，都可能照亮某个人的路。*
