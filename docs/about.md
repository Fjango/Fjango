# 📖 关于 Fjango

## 这是什么

Fjango 是一个**开放、免费、极简**的经验分享平台。

我们相信，每个人在学习和成长路上积累的经验都是有价值的。把这些经验分享出来，就能帮助到正在迷茫的后来者。

## 名字由来

**Fjango** = **F**ind **J**ourney **A**nd **N**avigate **G**rowth **O**nward

（发现旅程，导航成长）

## 设计理念

- **极简至上** — 白底黑字，去除一切不必要的元素，让内容自己说话
- **开放透明** — 全站开源，任何人都能看到网站的每一行代码
- **社区驱动** — 内容由社区贡献，人人都是作者

## 技术栈

本站使用以下开源技术构建：

- [MkDocs](https://www.mkdocs.org/) — 静态网站生成器
- [Material for MkDocs](https://squidfunk.github.io/mkdocs-material/) — 主题
- [GitHub Pages](https://pages.github.com/) — 免费托管
