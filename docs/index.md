# 欢迎来到 Fjango

**Fjango**，这是我的网名，没有任何实意。

这是一本大学生自救指南。

内容涉及：选专业、卷绩点、搞竞赛、找实习、考研、考公、留学申请、移民路径。每篇都写清楚适用条件、真实成本和风险 —— 不鼓吹任何一条路，只告诉你走这条路需要什么代价。

---

## 为什么搞

大学不教你怎么认识自己，只教你怎么做题。你从小到大都在闭卷考试，到了大学突然变成开卷人生，连题目都改成「你想成为谁」。不迷茫才怪。

迷茫的本质不是能力差，是你从来没被要求认真想过「我到底要什么」。

---

## 给谁看

在读大学生。尤其那些已经隐约觉得「按部就班不对」，但又不想随波逐流的人。

无论你是在纠结要不要出国、能不能留下来，还是只想先搞懂自己 —— 这里都有一份不带滤镜的参照。
